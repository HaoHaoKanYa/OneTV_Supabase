OneTV是一款基于Android TV的全新流媒体应用，使用Su云作为后端服务。本项目从1.41版本直接跳更至2.0.0，带来全新的用户体验和更稳定的性能。

OneTV仅为空壳软件，不提供任何直播源。用户需自行添加自定义直播源，并对使用内容负责。应用仅供个人学习和测试使用，请在24小时内删除。

## 系统要求

- Python 3.6或更高版本
- 无需额外依赖库
